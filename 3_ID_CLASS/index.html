ID e CLASS

ID

O ID funcionna como um CPF. Assim como só posso ter um numero de CPF, cada elemento só pode ter um valor de ID.o CPF é unico, ou seja, nunguem pode ter um CPF igual ao meu. Isso significa que o ID tambem é unico, então não é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona da maneira oposta ao ID. diferente do ID em que voce só pode ter um valor, no CLASS voce pode ter varios valores. Tambem diferente do ID, em que não se pode repetir um valor, no CLASS voce pode utilizar o mesmo valor para vários elementos.
